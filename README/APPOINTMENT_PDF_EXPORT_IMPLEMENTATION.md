## Data Flow

```
User clicks "Export Data"
    ↓
Fetch all filtered appointments (status, date range)
    ↓
Apply client-side filters (search, pet type, breed)
    ↓
Sort appointments (by booked date)
    ↓
Fetch clinic details (address, logo URL)
    ↓
Download logo image from Firebase Storage
    ↓
Get current user name for "Generated by"
    ↓
Generate PDF with AppointmentPdfService
    ↓
Download PDF to user's device
```

## Code Examples

### Fetching Clinic Logo
```dart
// Get clinic details
final clinicDetails = await ClinicDetailsService.getClinicDetails(clinicId);

// Download logo if available
if (clinicDetails.logoUrl != null) {
  final response = await http.get(Uri.parse(clinicDetails.logoUrl!));
  if (response.statusCode == 200) {
    clinicLogo = response.bodyBytes;
  }
}
```

### Generating PDF
```dart
final pdfBytes = await AppointmentPdfService.generateAppointmentReport(
  appointments: allAppointments,
  clinicName: clinicName,
  clinicAddress: clinicAddress,
  clinicLogo: clinicLogo,
  statusFilter: selectedStatus != 'All Status' ? selectedStatus : null,
  searchQuery: searchQuery.isNotEmpty ? searchQuery : null,
  startDate: startDate,
  endDate: endDate,
  petTypeFilter: selectedPetType,
  breedFilter: selectedBreed,
  generatedBy: adminName,
);
```

### Downloading PDF
```dart
final filename = 'appointment_report_${clinicName}_${timestamp}.pdf';
file_downloader.downloadFile(filename, pdfBytes);
```

## Statistics Calculations

### Completion Rate
```dart
final completionRate = total > 0 
    ? ((completed / total) * 100).toStringAsFixed(1) + '%' 
    : '0%';
```

**Example:** 40 completed out of 125 total = 32.0%

### Cancellation Rate
```dart
final cancellationRate = total > 0 
    ? ((cancelled / total) * 100).toStringAsFixed(1) + '%' 
    : '0%';
```

**Example:** 8 cancelled out of 125 total = 6.4%

## Error Handling

### Missing Clinic Logo
```dart
// Graceful fallback - PDF generates without logo
if (clinicDetails.logoUrl == null || download fails) {
  clinicLogo = null; // PDF service handles null logo
}
```

### Missing Clinic Address
```dart
// Fallback to default text
clinicAddress = clinicAddress.isNotEmpty 
    ? clinicAddress 
    : 'Address not available';
```

### Missing User Name
```dart
// Fallback to default footer
generatedBy = generatedBy ?? null; // Shows "PawSense Appointment Report"
```

### Empty Results
```dart
if (allAppointments.isEmpty) {
  ScaffoldMessenger.showSnackBar(
    SnackBar(content: Text('No appointments to export')),
  );
  return; // Don't generate PDF
}
```
